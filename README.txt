AlphaCity Layout Kit
====================
將本套件解壓到你的專案根目錄後，確認：
- launchers/Check-Layout.cmd  可直接雙擊檢查
- .pre-commit-config.yaml     如使用 pre-commit：`pip install pre-commit && pre-commit install`
- tools/Register-LayoutCheckTask.ps1  註冊夜間排程：`pwsh -File tools/Register-LayoutCheckTask.ps1`
- .gitignore_additions__alphacity.txt 請將內容追加到專案的 .gitignore
- docs/CHANGELOG.md            版本註記可直接提交
